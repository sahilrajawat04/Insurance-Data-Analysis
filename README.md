# Insurance Data Analysis & Visualization using Pandas

This repository contains a modular Python solution for analyzing insurance claims datasets using Pandas and Matplotlib.

## Repository Structure
* `data_analysis_queries.py`: Core logic handling joins, data type conversions, conditional flagging, and row filtering.
* `insurance_visualizations.py`: Generates distributions, trend tracking bar charts, summary pivot tables, and market segment pie charts.
* `claims.csv`, `cust_demographics.csv`, `DataAudit.csv`: Underlying project source datasets.
